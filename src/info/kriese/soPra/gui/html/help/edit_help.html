Daten bearbeiten

Ermöglicht das Bearbeiten vorhandener Daten oder auch das Anlegen eines neuen LOP (siehe Menü "Daten Bearbeiten").


Szene zeigen

Zeigt das LOP in der 2. geometrischen Interpretation in einer 3D-Ansicht.
Dabei wird sowohl der konvexe Kegel angezeigt, der den Lösungsraum aufspannt, als auch der Strahl, der den Lösungsbereich repräsentiert.


Lösung zeigen

Zeigt die berechnete Lösung des LOP in der 3D-Ansicht an.
Dabei wird der Lösungspunkt und Hilfslinien sowie das Koordinatensystem des LOP eingeblendet.


Duales Problem zeigen

Zeigt das Duale Problem zum aktuell bearbeiteten LOP an.

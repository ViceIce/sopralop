Daten bearbeiten

Hier können die Daten zu einem LOP in der 2. geometrischen Interpretation bearbeitet werden. Als Minimum sind hier drei Variablen vorgesehen, da LOP mit zwei Variablen und zwei Nebenbedingungen leichter in der 1. geometrischen Interpretation bzw. über ein einfaches Gleichungssystem gelöst werden können.
In die Felder der Tabelle werden die Koeffizienten der xi der Zielfunktion und der Nebenbedingungen eingegeben. Variablen, die in der Ziel- oder Nebenfunktion nicht vertreten sind, werden mit dem Koeffizienten Null angegeben.
Die Operanden (außer in der Zielfunktion) werden über PullDown-Menüs ausgewählt; die Auswahl, ob das Problem zu minimieren oder zu maximieren ist, muß als Text (die Kurzform "min" oder "max" genügt) eingegeben werden.
Dieser Menüpunkt kann auch im Verlauf der Bearbeitung eines LOP jederzeit aufgerufen werden.


Add Var.

Hier können Felder für zusätzliche Variablen (insgesamt maximal 10 Spalten) angelegt werden.
Die Felder werden als Spalte mit dem Index des nächsthöheren xi vor den Operanden eingefügt.


Delete Var.

Hier können Variablenfelder entfernt werden.
Gelöscht wird bei jeder Betätigung des Button jeweils die Spalte mit dem höchsten Index xi.
Die anderen Felder bleiben von der Anderung unberührt. Die ersten drei Spalten können nicht entfernt werden.


Clear Table

Hiermit wird der Inhalt der gesamten Tabelle gelöscht. Alle Koeffizienten werden auf Null gesetzt, die Operanden werden auf "=" gesetzt, und die Anzahl der Variablen wird auf drei (s.o.) reduziert.
ACHTUNG: Dieser Schritt kann nicht rückgängig gemacht werden! Ungesicherte Daten sind damit verloren!


Quit

Hiermit werden die eingegeben Daten in das LOP übernommen und das Eingabefenster geschlossen.
ACHTUNG: Solange in den Feldern ungültige Daten stehen (Rahmen des Feldes ist rot markiert), kann das Fenster nicht geschlossen werden, und die Daten werden nicht übernommen!

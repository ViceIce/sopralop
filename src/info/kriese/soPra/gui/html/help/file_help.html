Datei Öffnen

Ermöglicht das Laden eines zuvor gespeicherten LOP. Die LOP's sollten die Dateiendung ".lop" tragen und müssen im XML-Format gespeichert sein.


Datei Speichern

Ermöglicht das Speichern eines angelegten/bearbeiteten LOP zur weiteren Bearbeitung. Als Dateiendung ist ".lop" vorgesehen, die Daten werden in der Datei im XML-Format abgelegt.


Datei Speichern unter

Ermöglicht das Speichern eines angelegten/bearbeiteten LOP unter Angabe eines Dateinamens.


Beispiele

Hier können vordefinierte, spezielle Fälle von LOP's aufgerufen und bearbeitet werden.
Betrachtete Fälle:
- LOP hat keine Lösung
- Lösung des LOP streift eine Kante des Kegels
- LOP hat als Lösung eine Linie
- LOP hat als Lösung einen Strahl
- LOP hat ein Maximum, aber ist nach unten unbeschränkt
- LOP hat ein Minimum, aber ist nach oben unbeschränkt


Beenden

Beendet das Programm.

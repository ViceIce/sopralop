Allgemeines

Hier können die Daten zu einem LOP in der 2. geometrischen Interpretation bearbeitet werden. Als Minimum sind hier drei Variablen vorgesehen, da LOP mit zwei Variablen und zwei Nebenbedingungen leichter in der 1. geometrischen Interpretation bzw. über ein einfaches Gleichungssystem gelöst werden können. In die Felder der Tabelle werden die Koeffizienten der x(i) der Zielfunktion und der Nebenbedingungen eingegeben. Variablen, die in der Ziel- oder Nebenfunktion nicht vertreten sind, werden mit dem Koeffizienten Null angegeben. Die Auswahl, ob das Problem zu minimieren oder zu maximieren ist,wird über ein PullDown-Menü ausgewählt. Dieser Menüpunkt kann auch im Verlauf der Bearbeitung eines LOP jederzeit aufgerufen werden.


 

Variable hinzufügen

Hier können Felder für zusätzliche Variablen (insgesamt maximal 10 Spalten) angelegt werden. Die Felder werden als Spalte mit dem Index des nächsthöheren x(i) vor der Operanden-Spalte eingefügt.


 

Variable entfernen

Hier können Variablenfelder entfernt werden. Gelöscht wird bei jeder Betätigung des Button jeweils die Spalte mit dem höchsten Index x(i). Die anderen Felder bleiben von der Anderung unberührt. Die ersten drei Spalten können nicht entfernt werden.


 

Werte zurücksetzen

Hiermit werden Änderungen im Eingabefenster, welche noch nicht mit der Funktion "Werte übernehmen" in das LOP übernommen wuren, verworfen. Das Eingabefenster wird bis auf die letzten übernommenen Daten zurückgesetzt. ACHTUNG: Dieser Schritt kann nicht rückgängig gemacht werden!


 

Werte löschen

Hiermit wird der Inhalt der gesamten Tabelle gelöscht. Alle Koeffizienten werden auf Null gesetzt, die Operanden werden auf "=" gesetzt, und die Anzahl der Variablen wird auf drei (s.o.) reduziert. ACHTUNG: Dieser Schritt kann nicht rückgängig gemacht werden! Ungesicherte Daten sind damit verloren!


Werte übernehmen

Die eingegebenen Werte werden in das Problem übernommen. Das neue LOP wird automatisch mit den eigegebenen Werten gelöst. ACHTUNG: Wenn es Felder mit ungültigen Werten gibt (Rahmen des Feldes ist rot), wird eine Fehlermeldung erscheinen, und die Daten werden nicht in das LOP übernommen!


 

Lösung überprüfen

Mit dieser Funktion wird nach Eingabe der selbst ermittelten Lösung diese auf Korrektheit überprüft. Falls keine Lösung existiert, ist als Wert jeweils "ne" einzugeben. Bei unbeschränkten LOP's ist "u" einzugeben (jeweils ohne "")

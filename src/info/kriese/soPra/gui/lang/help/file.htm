  LOP öffnen

Ermöglicht das Laden eines zuvor gespeicherten LOP's. Die zu ladenden Dateien sollen die Dateiendung ".lop" tragen und müssen im XML-Format gespeichert sein.


 

  LOP speichern

Ermöglicht das Speichern eines angelegten/bearbeiteten LOP's zur weiteren Bearbeitung. Als Dateiendung ist ".lop" vorgesehen, die Daten werden in der Datei im XML-Format abgelegt.


 

  LOP speichern unter

Ermöglicht das Speichern eines angelegten/bearbeiteten LOP's unter Angabe eines Dateinamens.


 

Beispiel LOP's

Hier können vordefinierte, spezielle Fälle von LOP's aufgerufen und bearbeitet werden.
Betrachtete Fälle:

  • LOP hat genau eine optimale Lösung
  • LOP hat eine optimale Lösung, der Lösungsbereich des LOP ist unbeschränkt
  • LOP hat mehrere optimale Lösungen (Kante)
  • LOP hat keine Lösung
  • LOP hat keine Lösung, der Lösungsbereich ist unbeschränkt
  • LOP hat mehrere optimale Lösungen, der Lösungsbereich ist unbeschränkt

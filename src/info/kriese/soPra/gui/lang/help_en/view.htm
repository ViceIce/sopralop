General

Some Hints for using the 3D-View:
By pressing and holding the left mousebutton the coordinate system can be rotated around all axes.
By pressing and holding the right mousebutton the coordinate system can be relocated along the x- and y-axis (in relation to the view of the user).
Turning the mousewheel relocates the coordinate system along the z-axis (in relation to the view of the user).


 

  Reset 3D-Visualization (F2)

This button switches the 3D-visualization back to the original state. The coordinate system will be centered and the axes will be aligned correctly.


 

  3D-View change background color (F3)

With this function it is possible to change the background color of the 3D-View to white. This might be helpful for exporting a picture of the 3D-View or for taking a screenshot.
For a better visibility the cone will be repainted with high-contrast colors.


 

  Show 3D-Fullscreen-Window (F5)

Here the 3D-visualization is switched to a fullscreen window separated from the main program window, a more detailed view on the LOP is generated.


 

  Show Dual LOP (F7)

The dual problem of the actual edited LOP is shown.


 

  Show Primal LOP (F7)

The primal LOP of the actual edited LOP is shown.

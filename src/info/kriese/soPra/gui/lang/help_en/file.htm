  Open LOP (Ctrl+O)

Here you can load a formerly saved LOP. The files should carry the suffix ".lop" and must be saved as XML.


 

  Save LOP (Ctrl+S)

Here you can save a built/edited LOP for later use. The files should carry the suffix ".lop" and will be saved as XML.


 

  Save LOP as (Ctrl+Shift+S)

Option for saving a built/edited LOP for later use with the possibility to give an alternative filename.

LOP examples

Here you can find and edit a set of predefined special cases of LOP's.
Special cases:

  • LOP has a simgle optimal solution
  • LOP has an optimal Solution, the solution area is unlimited
  • LOP has multiple solutions (edge)
  • LOP has no solution
  • LOP has no optimal solution, the solution set is unlimited
  • LOP has more than one optimal solution, the solution set is unlimited


 

  Exit

Quits the application.
